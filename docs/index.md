# Spartan Governance Landing

Welcome to the Spartan Governance Landing application documentation.

## Overview

This application is built with React and Vite, providing a modern, fast development experience for AI-powered governance solutions.

## Features

- **AI Studio Integration**: Built with Google Gemini API
- **Stripe Payment Processing**: Complete payment infrastructure
- **Modern Tech Stack**: React 19, TypeScript, Vite
- **Cloud Deployment**: GCP and Railway integration
